# BLab Notify – Simple FiveM Notification

![preview](https://github.com/Bast1enFR/Blab_Notify/blob/main/view.png)

Unlimited stack • Custom sound • Matte black design • Standalone 0 Dependency!

### Installation (30 seconds)
1. Drag the blab_notify folder into your resources  
2. Add to your server.cfg → ensure blab_notify  
3. Restart or refresh + start  
→ That’s it!

### Usage (2 seconds)
```lua
exports.blab_notify:Notify("Title", "Description", "Type", (optional) duration in ms)
exports.blab_notify:Notify("BLab", "You just won $10,000", "success")
exports.blab_notify:Notify("Bank", "Withdrawal denied", "error", 8000)
exports.blab_notify:Notify("Info", "Server will restart in 5 min", "warning")
```
